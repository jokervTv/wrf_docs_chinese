    地图投影，科里奥利和曲率项

    jokervTv 1/14/2020 translateWRF

    ARW 求解器目前支持四种球体投影 —— 兰伯特正形投影，极地投影，墨卡托投影和经纬度投影。这些投影在 Haltiner 和 Williams（1980）中有所描述。对于其中的三个投影 —— 兰伯特正形投影，极地投影和墨卡托投影，变换是各向同性的。各向同性变换要求在网格的任何地方都要求 (Δx/Δy)∣earth=(\Delta x / \Delta y)|_{earth} =(Δx/Δy)∣earth​= 常数。 在以前的 ARW 版本中仅支持各向同性转换。 从 ARW 第 3 版开始，我们现在支持各向异性投影，在这种情况下为纬度-经度网格，并支持完整的纬度-经度全球模型。ARW 使用地图因子来实现投影，ARW V3 中引入的对各向异性变换的一般化要求从计算空间到物理空间的变换的 xxx 和 yyy 分量都必须有地图因子，以便适应各向异性。

    在 ARW 的计算空间中，Δx\Delta xΔx 和 Δy\Delta yΔy 是常数。球体的正交投影要求投影中网格点之间的物理距离随网格上的位置而变化。为了变换控制方程，将地图比例因子 mxm_xmx​ 和 mym_ymy​ 定义为计算空间中的距离与地球表面上相应距离的比值：

    (mx,my)=(Δx,Δy)DistanceOnTheEarth.(2.17)(m_x,m_y) = \frac{(\Delta x,\Delta y)}{DistanceOnTheEarth}. \tag{2.17} (mx​,my​)=DistanceOnTheEarth(Δx,Δy)​.(2.17)

    ARW 求解器通过以下方式重新定义动量变量，将比例尺因子包含在控制方程中

    U=μdu/my,V=μdv/mx,W=μdw/my,Ω=μdω/my.U = \mu_d u/m_y, V = \mu_d v/m_x, W = \mu_d w/m_y, \Omega = \mu_d \omega /m_y . U=μd​u/my​,V=μd​v/mx​,W=μd​w/my​,Ω=μd​ω/my​.

    利用这些重新定义的动量变量，可以将包括比例尺因子在内的主要预测方程（2.8）-（2.14）表示为

    ∂tU+mx[∂x(Uu)+∂y(Vu)]+∂η(Ωu)+(mx/my)[μdα∂xp+(α/αd)∂ηp∂xϕ]=FU(2.18)\begin{aligned} \partial_t U + m_x[\partial_x(Uu) + & \partial_y(Vu)] \\ & + &\partial_\eta(\Omega u) + (m_x/m_y)[\mu_d \alpha \partial_x p + (\alpha / \alpha_d)\partial_\eta p \partial_x \phi] = F_U \tag{2.18} \end{aligned} ∂t​U+mx​[∂x​(Uu)+​∂y​(Vu)]+​∂η​(Ωu)+(mx​/my​)[μd​α∂x​p+(α/αd​)∂η​p∂x​ϕ]=FU​​(2.18)

    ∂tV+my[∂x(Uv)+∂y(Vv)]+(my/mx)∂η(Ωv)+(my/mx)[μdα∂yp+(α/αd)∂ηp∂yϕ]=FV(2.19)\begin{aligned} \partial_t V + m_y[\partial_x(Uv) + & \partial_y(Vv)] \\ &+ (m_y/m_x)\partial_\eta(\Omega v) + (m_y/m_x)[\mu_d \alpha \partial_y p + (\alpha / \alpha_d)\partial_\eta p \partial_y \phi] = F_V \tag{2.19} \end{aligned} ∂t​V+my​[∂x​(Uv)+​∂y​(Vv)]+(my​/mx​)∂η​(Ωv)+(my​/mx​)[μd​α∂y​p+(α/αd​)∂η​p∂y​ϕ]=FV​​(2.19)

    ∂tW+mx[∂x(Uw)+∂y(Vw)]+∂η(Ωw)−my−1g[(α/αd)∂ηp−μd]=FW(2.20)\partial_t W + m_x[\partial_x(Uw) + \partial_y(Vw)] + \partial_\eta(\Omega w) - m^{-1}_y g [(\alpha / \alpha_d)\partial_\eta p - \mu_d] = F_W \tag{2.20} ∂t​W+mx​[∂x​(Uw)+∂y​(Vw)]+∂η​(Ωw)−my−1​g[(α/αd​)∂η​p−μd​]=FW​(2.20)

    ∂tΘm+mxmy[∂x(UΘm)+∂y(VΘm)]+my∂η(ΩΘm)=FΘm(2.21)\partial_t \Theta_m + m_x m_y[\partial_x(U\Theta_m) + \partial_y(V\Theta_m)] + m_y\partial_\eta(\Omega\Theta_m) = F_{\Theta_m} \tag{2.21} ∂t​Θm​+mx​my​[∂x​(UΘm​)+∂y​(VΘm​)]+my​∂η​(ΩΘm​)=FΘm​​(2.21)

    ∂tμd+mxmy[Ux+Vy]+my∂η(Ω)=0(2.22)\partial_t\mu_d + m_x m_y[Ux + Vy] + m_y\partial_\eta(\Omega) = 0 \tag{2.22} ∂t​μd​+mx​my​[Ux+Vy]+my​∂η​(Ω)=0(2.22)

    ∂tϕ+μd−1[mxmy(U∂xϕ+V∂yϕ)+myΩ∂ηϕ−mygW]=0(2.23)\partial_t\phi + \mu^{-1}_d [m_x m_y(U\partial_x\phi + V \partial_y\phi) + m_y\Omega\partial_\eta\phi-m_ygW] = 0 \tag{2.23} ∂t​ϕ+μd−1​[mx​my​(U∂x​ϕ+V∂y​ϕ)+my​Ω∂η​ϕ−my​gW]=0(2.23)

    ∂tQm+mxmy[∂x(Uqm)+∂y(Vqm)]+my∂η(Ωqm)=FQm(2.24)\partial_t Q_m + m_x m_y[\partial_x(Uq_m) + \partial_y(Vq_m)] + m_y\partial_\eta(\Omega q_m) = F_{Q_m} \tag{2.24} ∂t​Qm​+mx​my​[∂x​(Uqm​)+∂y​(Vqm​)]+my​∂η​(Ωqm​)=FQm​​(2.24)

    它们与诊断方程（2.15）和（2.16）一起求解。

    动量方程（2.18）–（2.20）的右侧项包含科里奥利和曲率项以及混合项和物理强迫。包括地图比例因子（2.17），科里奥利和曲率项以以下形式转换：

    FUcor=+mxmy(f+umymx∂mx∂y−v∂my∂x)V−(ure+ecos⁡αr)W(2.25)F_{U_{cor}} = + \frac{m_x}{m_y}(f + u\frac{m_y}{m_x} \frac{\partial m_x}{\partial y} - v \frac{\partial m_y}{\partial x})V - (\frac{u}{r_e} + e\cos\alpha_r)W \tag{2.25} FUcor​​=+my​mx​​(f+umx​my​​∂y∂mx​​−v∂x∂my​​)V−(re​u​+ecosαr​)W(2.25)

    FVcor=−mymx[(f+umymx∂mx∂y−v∂my∂x)U+(ure−esin⁡αr)W](2.26)F_{V_{cor}} = - \frac{m_y}{m_x}[(f + u\frac{m_y}{m_x} \frac{\partial m_x}{\partial y} - v \frac{\partial m_y}{\partial x})U + (\frac{u}{r_e} - e\sin\alpha_r)W] \tag{2.26} FVcor​​=−mx​my​​[(f+umx​my​​∂y∂mx​​−v∂x∂my​​)U+(re​u​−esinαr​)W](2.26)

    FWcor=+e(Ucos⁡αr−mxmyVsin⁡αr)+1re(uU+mxmyvV)(2.27)F_{W_{cor}} = + e(U\cos\alpha_r - \frac{m_x}{m_y}V\sin\alpha_r) + \frac{1}{r_e}(uU + \frac{m_x}{m_y}vV) \tag{2.27} FWcor​​=+e(Ucosαr​−my​mx​​Vsinαr​)+re​1​(uU+my​mx​​vV)(2.27)

    其中 αr\alpha_rαr​ 是 y 轴与子午线之间的局部旋转角，ψ\psiψ 是纬度，f=2Ωesin⁡ψf = 2\Omega_e\sin\psif=2Ωe​sinψ，e=2Ωecos⁡ψe = 2\Omega_e\cos\psie=2Ωe​cosψ，Ωe\Omega_eΩe​ 是地球的旋转角速度，rer_ere​ 是地球半径。在此公式中，我们将距地球中心的径向距离近似为平均地球半径 rer_ere​，并且没有考虑水平网格距离随半径的变化。包含比例尺因子的项代表水平曲率项，包含 rer_ere​ 的项代表垂直（地球表面）曲率，具有 eee 和 fff 的项代表科里奥利力。

    对于各向同性的投影（兰伯特正形投影，极地投影和墨卡托投影），地图比例因子在两个水平方向上都是相同的，因此 mx=my=mm_x = m_y = mmx​=my​=m，其中 mmm 通常仅随纬度变化。对于各向异性纬度-经度网格，mx=sec⁡ψm_x = \sec\psimx​=secψ 以及 my=1m_y = 1my​=1，假设 ∂/∂y=(1/re)∂/∂ψ\partial/\partial y = (1/r_e)\partial/\partial\psi∂/∂y=(1/re​)∂/∂ψ，则 ∂mx/∂y=(1/re)sec⁡ψtan⁡ψ\partial m_x/ \partial y = (1/r_e)\sec\psi\tan\psi∂mx​/∂y=(1/re​)secψtanψ 以及 ∂my/∂x=0\partial m_y/\partial x = 0∂my​/∂x=0。对于笛卡尔网格上的理想化情况，应指定地图比例因子 mx=my=1m_x = m_y = 1mx​=my​=1 ，指定 fff，并且 eee 和 re−1r^{-1}_ere−1​ 应为零以去除曲率项。
